# PMA5-123-3W--DC-Bias

 DC bias PCB for PMA5-123-3W MMIC 3W 10GHz
 - ATtiny412 SW controlled
 - LDO for VDD, typical 7V
 - Switch to switch off or on Vdd
 - Idd Current measurment
 - DAC controlled (negative) Vgg synthersizer
 - Inputs:
	+8 .. 12V DC
	PTT
 - Outputs
	Vdd
	Vgg (-1.% .. -0.75V
	
 
# Key components:

Attiny412
MIC29302
R05C1TF05S-R
INA168
